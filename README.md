### INSTALLATION AND START PROGRAM
#### installing code here
```
apt update
apt upgrade
apt install git
git clone https://github.com/Craglitch/MDItermux
chmod 777 install,sh
./install.sh
```
#### start program or usage
```
./donut
```
or you can move donut it to `$PREFIX/bin`
```
mv donut $PREFIX/bin
donut
```
